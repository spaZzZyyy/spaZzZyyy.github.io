Nigel the Mad Conductor

Click Me to Play!
Made in Unity Game Engine
Developed in a team of 4 Click to learn more!

Nigel is a 2D platformer, Nigel is a mouse with the ability to conduct to change the tempo of the music affecting the tempo of the platforms of his surrounding environment. Nigel also has the ability to enter MAD mode changing his environment to switch the state of the platforms in the environment.


Project Blue

Click Me to Play!
W A D to move Space to dash
Coded using Unity game engine

GitHub Tool Documentaion

Project Blue is a grey box game that was designed to quickly iterate seemless and fluid mechanics for platformer games. The code was designed in a way that is modular, so it's quick to change aspects of the game such as world physics and character traits such as speed and jump height. In continuing to polish the game it is to be noted that tweaking mechanics to correct values and creating playable maps, animations, and sprites are to be added. Documented changes with each push is recorded on it's github page that can be found in the link github button above.

Moon Base


Made using Blender

Documentation

The Moon Base project was designed in blender. The goal of the project was to create a base on the moon, it was designed in a low poly style to minimize resources necessary to incorporate it into a game. Through out the project various techiques in blender was utilized including light mapping and fluid dynamic animations. A full documentation can be found above.
